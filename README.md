# Onesearch #Search everything,
#from one place
◎ OneSearch

Privacy-first universal personal search engine. Runs 100% in your browser.

Search across Google Drive, Gmail, Google Photos, Google Sheets, and local files — all from a single search bar. No servers. No data leaves your device. No tracking. Ever.


✦ Features


Universal search — Drive, Gmail, Photos, Sheets, and local files in one place
Multi-account — connect multiple Google accounts simultaneously
BM25 ranking — production-grade relevance scoring with field boosts
Fuzzy search — typo-tolerant matching with Levenshtein distance
Exact phrase — wrap in quotes "like this" for precise matches
Prefix commands — gmail:invoice, drive:report, photos:beach, sheet:budget
Live suggestions — dropdown suggestions as you type, with category grouping
"Did you mean?" — automatic typo correction banner on low-result queries
Smart filters — by type, service, account, date range, and sort order
Starred items — star anything, accessible in the Starred view
Preview panel — inline preview for images, PDFs, audio, video, markdown, CSV, JSON, code
Virtual scrolling — renders thousands of results without lag
Dark / light theme — system preference detected, manually toggleable
Local files — index folders from your device via File System Access API
Background indexing — index status bar, no blocking
Search history — recent searches, pin favourites, clear any time
100% offline after indexing — search works without internet
